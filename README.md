# Homebrew Tap for mac-cleanup

This repository is a Homebrew tap for the [mac-cleanup](https://github.com/PiusSunday/mac-cleanup) CLI tool.

## Installation

To install `mac-cleanup` using this tap, run:

```bash
brew install PiusSunday/mac-cleanup/mac-cleanup
```

## Development

To install the formula locally for development, you can use `brew install --HEAD`:

```bash
brew install --HEAD ./Formula/mac-cleanup.rb
```
